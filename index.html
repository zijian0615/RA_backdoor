BackdoorRA:

A practical backdoor attack against Robotic Arm

Hardware

BackdoorRA is a practical backdoor attack against all robotic arms system that can be used to manipulate the robot's behavior in a targeted manner.

System Overview

We propose a three-stage attack framework comprising: attack module identification, poisoned data generation, and backdoor injection. The system identifies control modes and attack windows based on action-observation sequences, and defines attack templates (pose, speed, force deviations) aligned with task keyframes. Counterfactual samples are generated by modifying motor rotations and embedding visual triggers, which are placed strategically to evade detection. Poisoned and clean data are used to retrain or fine-tune the original model without changing the training routine. The final backdoor model is then deployed to a real-world testbed for evaluation.

Experiments

(1) Soft object handing

Task The robotic arm picks up a sponge and places it into a transparent container. It must apply a moderate gripping force that is sufficient to securely grasp the soft object without causing any deformation or damage.
Target The attack manipulates the gripper motor by reducing its rotation angle, which increases the applied gripping force. As a result, the soft object undergoes significant deformation, potentially simulating damage or operational failure.
